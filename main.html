Magic Proxies

Instructions:
Just input the cards you'd like to proxy in the following format:
X CARD_NAME

Where X is the number of the proxies you'd like (defaults to 1)
and CARD_NAME is any valid Magic: The Gathering Card name

If your card isn't showing up its probably because of
a typo in the name or in the formatting

Special Thanks to Scryfall.com for creating a wonderful API that makes all this possible
